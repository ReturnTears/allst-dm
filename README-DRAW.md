# 大话设计模式
```text
依赖（Dependency） ：带虚线箭头表示
关联（Association）：带实线箭头表示
聚合（Aggregation）：带空心菱形箭头的实线表示
组合（Composition）：带实心菱形箭头的实线表示
实现（Realization）：带空心箭头和虚线表示
继承（Inheritance）：带空心箭头的实线表示
```

## 策略模式
```text
策略模式（Strategy Pattern）是一种行为设计模式，它允许你定义一系列的算法，并将它们封装起来。策略模式使你可以在运行时切换这些算法。
它定义了算法家族，分别封装起来，让他们之间可以相互替换，此模式让算的变化，不会影响到使用算法的客户。



```
